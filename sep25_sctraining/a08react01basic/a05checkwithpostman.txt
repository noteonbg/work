 Enhanced Example with Headers in Postman

We'll continue using JSONPlaceholder with a POST request.


Step-by-Step Instructions

1. Set Up POST Request

 Method: `POST`
 URL: `https://jsonplaceholder.typicode.com/posts`

---

# 2. Add Request Headers

Click the "Headers" tab in Postman, and add the following:

| KEY             | VALUE                    |
| --------------- | ------------------------ |
| Content-Type    | application/json         |
| X-Custom-Header | CustomHeaderValue        |
| Authorization   | Bearer dummy\_token\_123 |

# 4. Send the Request

Click Send.


# 5. View Response Headers

After you get a response:

 Scroll down to the "Headers" section of the response.
 You'll see headers like:


Content-Type: application/json; charset=utf-8
Content-Length: 83
Connection: keep-alive
X-Powered-By: Express


These are the response headers sent by the server.
