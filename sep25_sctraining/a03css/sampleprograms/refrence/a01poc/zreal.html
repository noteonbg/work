Fixed-width Container

This container has a fixed width, and it will resize at different screen sizes.

Full-width Container

This container spans the entire width of the viewport, regardless of screen size.

Small Screen Responsive Container

This container will have a maximum width of 540px on small screens and automatically adjust on larger screens.

Medium Screen Responsive Container

This container has a maximum width of 720px on medium screens and adjusts based on the viewport size.

Large Screen Responsive Container

This container will have a maximum width of 960px on large screens.

Extra Large Screen Responsive Container

This container will have a maximum width of 1140px on extra-large screens.
