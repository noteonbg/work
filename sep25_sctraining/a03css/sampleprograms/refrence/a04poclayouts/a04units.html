Pixel (px)

200px x 100px (Fixed size in pixels)

The pixel (px) is a fixed unit used for precise layout dimensions. It doesn't change based on screen size or resolution.

Percent (%)

50% width of its parent container

The percent (%) unit is relative to the parent element's dimensions. In this case, the width is 50% of the parent container's width.

Em (em)

Font size 4em (relative to its parent's font size)

The em unit is relative to the font size of the parent element. Here, the font size is 2 times the parent element's font size.

Rem (rem)

Font size 2rem (relative to the root font size)

The rem unit is relative to the root element's font size (typically html). In this case, it is 2 times the root font size.

Viewport Width (vw)

50vw (50% of the viewport's width)

The vw (viewport width) unit is relative to the width of the viewport. Here, the element is 50% of the viewport's width.

Viewport Height (vh)

50vh (50% of the viewport's height)

The vh (viewport height) unit is relative to the height of the viewport. Here, the element is 50% of the viewport's height.

Viewport Minimum (vmin)

50vmin (50% of the smaller dimension of the viewport)

The vmin unit is relative to the smaller of the viewport's width or height. It helps ensure that the element scales proportionally across different screen sizes.

Viewport Maximum (vmax)

50vmax (50% of the larger dimension of the viewport)

The vmax unit is relative to the larger of the viewport's width or height. It ensures that the element scales based on the largest dimension of the viewport.

Auto

Width and height set to 'auto'

The auto unit is used to allow the browser to determine the width or height based on content, parent container, or other factors.

Ch (ch)

10ch (10 character widths)

The ch unit is based on the width of the "0" (zero) character in the font. It is often used for sizing input fields.
